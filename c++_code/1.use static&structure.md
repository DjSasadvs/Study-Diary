题目描述
--------
求1+2+3+...+n，要求不能使用乘除法、for、while、if、else、switch、case等关键字及条件判断语句（A?B:C），也不能使用公式。

```
class Solution {
public:
    static int sum;
    static int num;
    Solution(){
        num++;
        sum+=num;
    }

    int Sum_Solution(int n) {
        sum = num = 0;
        Solution *solu = new Solution[n];
        int result = sum;
        delete []solu;
        sum = 0;
        num =1;
        return result;
    }
};
int Solution::sum = 0;
int Solution::num = 0;
```
